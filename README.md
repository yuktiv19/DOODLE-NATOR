# 🎨 DOODLE-NATOR
**A Real-Time Sketch Recognition Engine**

DOODLE-NATOR is an AI-powered drawing game that uses a Convolutional Neural Network (CNN) to recognize sketches as you draw them. 

## 🚀 Features
- **Instant Recognition:** Predicts drawings in real-time.
- **Edge AI:** The model runs locally in the browser via TensorFlow.js (no server lag!).
- **Categories:** Recognizes Apples, Stars, Cats, Fish, and Houses.

## 🛠️ Tech Stack
- **Model:** TensorFlow/Keras (Python)
- **Deployment:** TensorFlow.js
- **Frontend:** HTML5, CSS3, JavaScript

## 📂 Structure
- `/website`: The web application and the AI model.
- `train_model.ipynb`: The Google Colab notebook used for training.
